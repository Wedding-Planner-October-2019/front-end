Wedding Planner (frontend)

Proposal: Our mission is to provide users with their dream wedding..

There are featured profiles to view and if interested user can register for their wedding.

Frameworks/Libraries Used :

React

Redux 
 
Axios

How to View The Project You can visit the link provided in the description, or, if you'd like to fiddle around with the project, you can follow these steps:

Clone this repo. 

Inside your terminal, run cd frontend.

Run npm i && npm start.

 Visit localhost:3000 to view the project.
